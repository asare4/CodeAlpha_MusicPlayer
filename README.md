# Music Player

![Music Player Screenshot](images/img10.jpg)

## Overview

This is a **JavaScript-based music player** with a sleek, modern UI built using HTML, CSS, and JavaScript.  
It allows users to **play, pause, navigate tracks, control volume, and track progress** with an interactive and responsive interface.

- **Volume control slider**  
- **Autoplay next track** after current track ends  
- Responsive design for **desktop and mobile devices**  
- **Glassmorphism UI** with smooth hover effects and shadows  
- Optional playlist support
